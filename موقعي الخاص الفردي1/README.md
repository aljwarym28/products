# Saqoor Store

نسخة أولى ثابتة لموقع عربي متخصص بمنتجات منصة MT5 الرقمية.

## الصفحات

- `index.html`: الصفحة الرئيسية، الأقسام، طريقة الطلب، الأسئلة، والتواصل.
- `products.html`: صفحة المتجر مع بحث وتصفية حسب نوع المنتج.
- `styles.css`: تنسيقات الواجهة.
- `script.js`: القائمة على الجوال، فلترة المنتجات، وروابط واتساب التلقائية.

## فكرة النسخة الحالية

المنتجات النهائية غير جاهزة بعد، لذلك الواجهة تعرض منتجات مقترحة بحالة "قريباً" وتستخدم واتساب لاستقبال الطلبات والاستفسارات.

## طريقة الفتح

افتح ملف `index.html` مباشرة في المتصفح لمعاينة الموقع.

## رقم واتساب المستخدم

`+964 781 557 6089`

عند تغيير الرقم لاحقاً، ابحث عن `9647815576089` داخل الملفات واستبدله بالرقم الجديد.

## تحميل المنتجات المجانية

ضع ملفات المنتجات داخل مجلد `downloads`.

الملفات الحالية:

```text
downloads/السيطرة على الذات قبل السوق.zip
downloads/كتاب خطة تحويل 10 دولار الى مليون دولار.zip
```

بعد رفع الملفات إلى Hostinger سيكون رابط التحميل:

```text
https://saqoorstore.com/downloads/السيطرة%20على%20الذات%20قبل%20السوق.zip
```

لجعل زر المنتج يحمل الملف مباشرة، استخدم رابط مثل:

```html
<a class="btn btn-card" href="downloads/السيطرة على الذات قبل السوق.zip" download>تحميل مجاني</a>
```
